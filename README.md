# CursoClaro

CursoClaro es una plataforma orientada a familias para organizar la comunicación escolar, la coordinación comunitaria del curso y, en fases posteriores, servicios y documentación asociada a cada alumno.

## Estructura inicial

- `docs/`: documentación funcional, técnica y de seguridad
- `assets/brand/`: assets oficiales de marca
- `assets/mockups/`: mockups y referencias visuales vigentes
- `archive/`: archivos descartados o no oficiales que se conservan por referencia
